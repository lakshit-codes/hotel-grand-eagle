# NestCraft Next.js Conversion

This project was converted from a Vite + React + TypeScript app to a Next.js App Router project.

## Run locally

```bash
npm install
npm run dev
```

## Routes

- /
- /shop
- /category/[id]
- /product/[id]
- /about
- /services
- /blog
- /faq
- /contact
- /cart
- /checkout
